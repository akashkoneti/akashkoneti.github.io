Proteomic Data Exploration Suite

This featured project demonstrates advanced proteomics data analysis using interactive Shiny visualizations. It includes PCA analysis, clustering, linear regression tests, statistical significance analysis, and visualization of p-values. The goal is to provide an interactive experience for exploring the relationships between proteins and gene expression levels, highlighting significant findings.

Analysis Highlights

Normalization

Normalization methods are applied to make samples comparable and remove batch effects.

Identifying Differentially Expressed Proteins

Statistical tests are used to find proteins with significant changes in abundance between groups (e.g., diseased vs healthy).

T-tests

Student's t-test or moderated t-tests (e.g., limma package) are applied to compare protein abundance between two groups.

ANOVA

Extends t-tests to compare more than two groups or experiments with multiple factors and interactions.

Fold Change Analysis

Measures the ratio of protein abundance between two conditions, often visualized alongside p-values.

Volcano Plots

Combines fold change and statistical significance to highlight potential differentially expressed proteins.

Multiple Testing Correction

Methods like FDR, Benjamini-Hochberg procedure, and Q-values are applied to control false positives.

Multivariate Analysis

PCA, clustering, and machine learning methods (Random Forest, SVM, Elastic Net) are applied for feature selection and predictive modeling.

Data Science Concepts Demonstrated

Platform Overview

  • Built with R Shiny & Plotly for interactive visualizations
  • Modular, tab-based UI for clean organization
  • Exportable plots as standalone HTML for reproducibility

Data Preprocessing

  • Filtering missing or zero values
  • Log₂ transformation for normalization
  • Reactive updates for instant plot rendering
  • Clear separation of filtering, plotting, and rendering logic

Analytical Concepts

  • Univariate analysis via histograms
  • Bivariate analysis with scatter & regression plots
  • Outlier detection using Volcano plots
  • Multivariate structure detection with PCA

Statistical Techniques

  • Linear regression (lm) with R² assessment
  • Hypothesis testing (p-values)
  • Fold change & log-scale visualizations
  • Multiple testing visualization
