Project Overview

This project demonstrates advanced data analysis techniques applied to proteomics and mRNA datasets. It includes PCA analysis, clustering, linear regression tests, statistical significance analysis, and visualization of p-values. The goal is to provide an interactive experience for exploring the relationships between proteins and gene expression levels, highlighting significant findings.

Key Skills Highlighted

  • Interactive dashboards with R Shiny
  • Dynamic visualizations with Plotly
  • Data preprocessing, cleaning, and transformation
  • Statistical analysis (regression, hypothesis testing)
  • Dimensionality reduction & multivariate analysis (PCA)
  • Reproducibility: saving interactive plots & structured pipeline

Analysis Highlights

Normalization

Normalization methods are applied to make samples comparable and remove batch effects.

Identifying Differentially Expressed Proteins

Statistical tests are used to find proteins with significant changes in abundance between groups (e.g., diseased vs healthy).

T-tests

Student's t-test or moderated t-tests (e.g., limma package) are applied to compare protein abundance between two groups.

ANOVA

Extends t-tests to compare more than two groups or experiments with multiple factors and interactions.

Fold Change Analysis

Measures the ratio of protein abundance between two conditions, often visualized alongside p-values.

Volcano Plots

Combines fold change and statistical significance to highlight potential differentially expressed proteins.

Multiple Testing Correction

Methods like FDR, Benjamini-Hochberg procedure, and Q-values are applied to control false positives.

Multivariate Analysis

PCA, clustering, and machine learning methods (Random Forest, SVM, Elastic Net) are applied for feature selection and predictive modeling.

Interactive Visualizations

1. Volcano Plot

Highlighting significant proteins based on fold change and p-values.

2. Regression Plot

Log2 regression analysis between proteome and transcriptome columns with R² metrics.

3. Histogram Plot

Distribution analysis of selected proteome and transcriptome columns.

4. Scatter Plot

Fold change scatter plot with threshold-based highlighting of proteins.

5. PCA Plot

Principal Component Analysis showing sample clustering and variation across conditions.
